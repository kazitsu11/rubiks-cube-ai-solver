# AI Rubik’s Cube Solver (C++)

This project implements a 2×2 Rubik’s Cube solver using Artificial Intelligence search algorithms.

Algorithms:
- Iterative Deepening DFS (IDDFS)
- A* Search
- IDA* (Korf’s Algorithm)

Concepts Used:
- State Space Search
- Heuristic Functions
- Recursion
- Optimization
- Graph Traversal

Language: C++
